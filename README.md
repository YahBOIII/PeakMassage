# Peak Recovery Massage Website

This repository contains a Next.js website for Peak Recovery Massage with:

- Multi-page marketing site (Home, About, Services, Book Appointment, Testimonials, Contact)
- Premium dark athletic theme with black, silver, and gold accents
- SEO metadata, Open Graph tags, LocalBusiness schema, FAQ schema, sitemap, and robots.txt

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run lint
npm run build
```
